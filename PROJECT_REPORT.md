# 宠物搜打撤 - 项目完成报告

## 项目概述
将宠物养成与搜打撤玩法结合的 2D 游戏，使用 Godot 4.6 开发。

---

## ✅ 已实现的核心功能

### 1. 界面流程（完整实现）

```
主菜单
├─ 开始游戏 → 创建新存档 → 准备界面
└─ 读取存档 → 选择存档 → 准备界面

准备界面
├─ 仓库（存储物资）
├─ 图鉴（宠物收集）
├─ 配装（宠物装备/技能）
└─ 行前准备 → 地图选择 → 出发

地图选择
└─ 史前世界 → 游戏场景

游戏场景
├─ 搜索物资点
├─ 击杀/收服敌人
├─ 拾取掉落物品
└─ 撤离点（10秒读条）→ 撤离成功 → 返回准备界面
```

### 2. 宠物系统（核心特色）

**宠物数据库**: 5种史前宠物
- 迅猛龙（常见，近战）
- 翼龙（稀有，远程）
- 剑齿虎（稀有，近战）
- 三角龙（史诗，近战）
- 霸王龙（传说，近战）

**宠物属性**:
- HP、攻击、防御、速度
- 技能系统（近战/远程攻击、Buff）
- 收服概率

**宠物合体系统**:
- 按 G 键与宠物合体
- 获得宠物属性和技能
- 合体持续时间限制

### 3. 战斗系统（无枪械）

**玩家攻击**:
- J 键普通攻击（伤害低）
- 合体后使用宠物技能（伤害高）

**敌人AI**:
- 自动追踪玩家
- 攻击冷却
- 基于宠物的属性

**收服系统**:
- 击败敌人有概率直接收服
- 未收服有概率掉落宠物蛋

### 4. 物资系统

**物资类型**:
- 材料（远古骨头、恐龙鳞片等）
- 装备（兽皮护甲、恐龙骨甲等）
- 消耗品（草药、治疗药剂等）

**物资点**:
- 3个等级（普通/中级/高级）
- 随机生成战利品
- 搜索后物品进入背包

### 5. 存档系统

- 3个存档槽位
- JSON 格式保存
- 保存内容：仓库、宠物图鉴、装备、游戏时长

## 🎮 操作说明

| 按键 | 功能 |
|------|------|
| **WASD** | 移动 |
| **F** | 搜索物资/拾取物品/撤离 |
| **J** | 攻击 |
| **G** | 宠物合体/解除合体 |

---

## 🎯 游戏流程

1. **主菜单** → 点击"开始游戏"
2. **准备界面** → 点击"行前准备"
3. **地图选择** → 点击"史前世界"，然后"出发"
4. **游戏场景**:
   - 移动到物资点，按 F 搜索
   - 遇到敌人，按 J 攻击
   - 按 G 与宠物合体（需要先在配装界面装备宠物）
   - 击败敌人有概率收服或获得宠物蛋
   - 拾取掉落物品
   - 移动到撤离点（绿色区域）
   - 按住 F 10秒完成撤离
5. **撤离成功** → 物品自动存入仓库
6. **返回准备界面** → 继续探险

## 📊 技术实现

### 全局单例（Autoload）

1. **GameData** - 游戏状态管理
2. **SaveSystem** - 存档读写
3. **PetDB** - 宠物数据库
4. **ItemDB** - 物品数据库

### 核心系统

- **宠物合体系统**: 动态改变玩家属性和技能
- **收服系统**: 概率收服 + 宠物蛋掉落
- **撤离读条**: 进度条显示，可中断
- **背包/仓库**: 双层物品存储系统
