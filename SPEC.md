> 版本：v1.0  
> 日期：2026-04-15  
> 项目：宠物搜打撤（Pet Extraction）  
> 引擎：Godot 4.6.1

---

## 1. 项目概述

### 1.1 游戏定位

搜打撤（搜索→战斗→撤离）玩法的宠物养成游戏。玩家在随机生成的地图中搜索物资、击败怪物、收集宠物，并在限定时间内抵达撤离点成功撤离。借鉴三角洲行动的物资管理和撤离机制，结合王者荣耀的技能系统，形成独特的游戏体验。

### 1.2 现有系统分析

| 模块 | 现状 | 目标 |
|------|------|------|
| 背包 | 纯单格10×6网格 | 保留单格，增加物资箱概念 |
| 仓库 | 简单数组存储 | 扩展为独立仓库界面 |
| 战斗 | 仅普攻 | 普攻 + 3个技能槽 |
| 地图 | 程序化生成 | 随机生成 + 路径验证 |
| 撤离 | 简单倒计时 | 完善撤离机制 |

## 2. 背包与仓储系统

### 2.1 格子系统设计

#### 2.1.1 背包（Inventory）

- **容量**：10列 × 6行 = 60格
- **单格规则**：每个格子可存放1个物品（不可堆叠）或同类物品堆叠
- **操作**：
  - 拖拽移动
  - 点击使用（消耗品立即生效）
  - 右键快捷使用

#### 2.1.2 仓库（Warehouse）

- **容量**：10列 × 8行 = 80格
- **解锁条件**：默认解锁第一个仓库页，后续页面可通过游戏内货币解锁
- **操作**：
  - 只能从背包移入，不能直接使用物品
  - 支持批量转移（选中多个物品）

#### 2.1.3 物资箱（Loot Box）

> 参考三角洲行动，但简化为单格系统

物资箱是玩家在地图中击杀怪物或开启物资点后获得的战利品容器。

**获取方式**：
1. 击杀怪物 → 掉落怪物箱（基于怪物类型）
2. 开启物资点 → 掉落物资箱（基于物资点等级）

**物资箱类型**：

| 类型 | 品质 | 颜色 | 说明 |
|------|------|------|------|
| 破旧箱 | 普通 | 灰色 | 低等级怪物/物资点 |
| 普通箱 | 优秀 | 蓝色 | 中等级怪物/物资点 |
| 精良箱 | 稀有 | 紫色 | 高等级怪物/物资点 |
| 珍贵箱 | 史诗 | 橙色 | Boss/高级物资点 |
| 传说箱 | 传说 | 红色 | 特殊事件 |

**物资箱UI**：
- 打开物资箱时弹出独立UI
- 显示箱内物品列表
- 支持"全部拾取"和"选择性拾取"
- 背包满时提示"部分物品无法拾取"

## 3. 技能与战斗系统

### 3.1 技能槽设计

参考王者荣耀，每个角色拥有：

- **普通攻击**：基础物理伤害
- **技能1（Q）**：冷却最短，消耗最低
- **技能2（W）**：冷却中等，效果辅助/控制
- **技能3（E）**：冷却最长，效果强大

> 不设置召唤师技能槽，简化操作

### 3.2 玩家技能系统

#### 3.2.1 技能数据结构

```gdscript
# 技能定义
{
    "id": "skill_001",
    "name": "利爪挥击",
    "icon": "skill_icon_001.png",
    "type": "melee",           # melee/ranged/buff/debuff/heal
    "damage": 50,              # 基础伤害
    "range": 80,               # 施放范围（远程技能有效）
    "radius": 0,               # AOE半径（0为单体）
    "cooldown": 3.0,           # 冷却时间（秒）
    "cost": 0,                 # 能量/法力消耗
    "duration": 0,             # 持续时间（buff/debuff有效）
    "effects": [],             # 附加效果列表
    "description": "挥动利爪对目标造成伤害"
}
```

#### 3.2.2 技能类型

| 类型 | 说明 | 示例 |
|------|------|------|
| melee | 近战单体攻击 | 利爪挥击、劈砍 |
| ranged | 远程单体/穿透攻击 | 风刃、箭雨 |
| buff | 增益效果 | 攻击力提升、攻速提升 |
| debuff | 减益效果 | 减速、中毒 |
| heal | 生命恢复 | 治疗术 |
| special | 特殊效果 | 位移、召唤 |

### 3.3 怪物技能系统

#### 3.3.1 怪物分类

| 分类 | 特点 | 示例 |
|------|------|------|
| 普通怪物 | 1-2个技能，近战 | 迅猛龙、剑齿虎 |
| 精英怪物 | 2-3个技能，有特殊效果 | 翼龙首领 |
| Boss | 3个技能 + 特殊机制 | 霸王龙 |

#### 3.3.2 怪物AI行为树

```
┌─────────────────────────────────────────┐
│              巡逻状态                   │
│    （随机移动，保持在出生点附近）        │
└──────────────┬──────────────────────────┘
               │ 发现玩家
               ▼
┌─────────────────────────────────────────┐
│              追击状态                   │
│    （向玩家移动，进入攻击范围）          │
└──────────────┬──────────────────────────┘
               │ 进入攻击范围
               ▼
┌─────────────────────────────────────────┐
│              战斗状态                   │
│  ┌─────────────────────────────────┐   │
│  │  普攻 ↔ 技能1 ↔ 技能2 ↔ 技能3   │   │
│  │  （根据AI优先级和冷却选择）      │   │
│  └─────────────────────────────────┘   │
└──────────────┬──────────────────────────┘
               │ 玩家死亡/逃离/消失
               ▼
┌─────────────────────────────────────────┐
│              返回状态                   │
│    （返回巡逻/追击状态）                 │
└─────────────────────────────────────────┘
```

### 3.4 伤害计算公式

```
最终伤害 = max(1, 基础伤害 × 技能系数 - 防御减免)

防御减免 = min(伤害上限, 防御力 × 防御系数)
- 物理防御系数：0.5
- 魔法防御系数：0.5
- 伤害上限：基础伤害的80%
```

## 4. 地图与撤离系统

### 4.1 地图生成系统

#### 4.1.1 地图参数

| 参数 | 数值 | 说明 |
|------|------|------|
| 地图宽度 | 20格 | |
| 地图高度 | 15格 | |
| 格子大小 | 64像素 | |
| 地形类型 | 7种 | 草地、墙体、道路、水域、沼泽、毒区、建筑 |

#### 4.1.2 地形类型

| 地形 | 代码 | 可通行 | 效果 |
|------|------|--------|------|
| 草地 | 0 | ✅ | 无 |
| 墙体 | 1 | ❌ | 阻挡 |
| 道路 | 2 | ✅ | 无 |
| 水域 | 3 | ✅(减速) | 移动速度-30% |
| 沼泽 | 4 | ✅(减速) | 移动速度-50%，每3秒-5HP |
| 毒区 | 5 | ✅(危险) | 每2秒-10HP |
| 建筑 | 6 | ❌ | 阻挡，可进入室内 |

#### 4.1.3 随机生成算法

```
1. 初始化网格（全草地）

2. 生成外圈围墙
   - 地图边缘强制为墙体

3. 生成道路网络（A*路径验证）
   - 确保玩家出生点到撤离点存在可达路径
   - 道路密度：15-25%

4. 随机生成地形区域
   - 水域：1-2个区域，每个3-5格
   - 沼泽：1-2个区域，每个2-4格
   - 毒区：1个区域，3-6格

5. 随机生成建筑
   - 2-4个建筑
   - 每个建筑4-9格（2×2 到 3×3）
   - 建筑内部随机生成1-2个家具/宝箱

6. 放置撤离点
   - 随机选择地图边缘的安全区域
   - 距离出生点至少10格

7. 放置物资点
   - 3-6个物资点
   - 分布：道路上40%，开阔区域60%

8. 生成怪物
   - 数量根据地图难度动态调整
   - 避开道路和物资点
   - 确保有至少一条安全路线
```

#### 4.1.4 路径验证算法

```gdscript
# 伪代码：A*路径验证
func validate_extraction_path(start: Vector2, end: Vector2) -> bool:
    # 1. 使用A*算法搜索路径
    path = a_star_search(start, end, walkable_tiles)
    
    # 2. 路径必须存在
    if path.is_empty():
        return false
    
    # 3. 路径长度不能超过阈值
    if path.length() > max_path_length:
        return false
    
    # 4. 路径上不能有无法通行的地形
    for tile in path:
        if not is_walkable(tile):
            return false
    
    return true
```

### 4.2 撤离系统

#### 4.2.1 撤离触发条件

- 玩家到达撤离点区域
- 撤离倒计时开始（默认3秒）
- 玩家在倒计时结束前保持在撤离区域内
- 成功：获得物资、经验，返回准备界面
- 失败（倒计时结束前离开/被击杀）：物资丢失，返回死亡界面

#### 4.2.2 撤离UI

```
┌─────────────────────────────────────────┐
│                                         │
│          🚁 撤离点                      │
│                                         │
│      ┌─────────────────────┐            │
│      │     ⏱️ 3.0 秒       │            │
│      │  ████████████░░░░   │            │
│      │   停留在圈内以撤离   │            │
│      └─────────────────────┘            │
│                                         │
│     [离开撤离点]  玩家在倒计时结束前     │
│                   离开撤离区域会取消撤离  │
└─────────────────────────────────────────┘
```

#### 4.2.3 撤离类型

| 类型 | 触发条件 | 奖励 |
|------|----------|------|
| 正常撤离 | 倒计时结束 | 100%物资 |
| 提前撤离 | 2秒后可提前撤离 | 70%物资 |
| 紧急撤离 | 特定条件触发 | 50%物资 |

### 4.3 撤离失败判定

以下情况判定为撤离失败：
1. 玩家死亡
2. 倒计时结束前离开撤离区域
3. 倒计时结束未到达撤离区域

### 5.2 各界面详细说明

#### 5.2.1 主菜单（MainMenu）

**入口**：
- 游戏启动

**内容**：
- 游戏标题 + 版本号
- 新游戏按钮
- 继续按钮（如果有存档）
- 设置按钮
- 退出按钮

**转场**：
- 新游戏 → 准备界面
- 继续 → 准备界面

#### 5.2.2 准备界面（PrepScreen）

**入口**：
- 主菜单 → 新游戏/继续
- 撤离成功后返回
- 撤离失败后返回

**功能**：
- 显示已收集宠物数量
- 显示仓库物品数量
- 显示游戏时长
- 显示已装备宠物
- 仓库管理按钮
- 宠物图鉴按钮
- 技能配置按钮（预留）
- 行前准备按钮 → 地图选择
- 保存进度按钮
- 返回主菜单按钮

**转场**：
- 行前准备 → 地图选择
- 返回 → 主菜单

#### 5.2.3 地图选择（MapSelect）

**入口**：
- 准备界面 → 行前准备

**内容**：
- 地图列表（史前世界、侏罗纪世界等）
- 难度选择（简单、普通、困难）
- 预计收益提示
- 出发按钮

**转场**：
- 出发 → 游戏世界
- 返回 → 准备界面

**状态重置**：
- 玩家血量回满
- 战斗时间归零
- 背包清空（物资点物品）
- 仓库保留

#### 5.2.4 游戏世界（GameWorld）

**入口**：
- 地图选择 → 出发

**HUD元素**：
- 玩家血条
- 能量条（扩展用）
- 技能图标 + 冷却指示
- 背包物品快捷栏
- 小地图（可选）
- 撤离倒计时（靠近撤离点时显示）

**操作**：
- WASD/方向键：移动
- 鼠标左键：普通攻击/点击交互
- Q/W/E：释放技能
- Tab：打开背包
- F：交互（拾取、使用）
- ESC：暂停菜单

**转场**：
- 死亡 → 撤离失败
- 撤离成功 → 撤离成功界面

#### 5.2.5 撤离成功（ExtractSuccess）

**入口**：
- 玩家在撤离点停留足够时间

**内容**：
- 撤离成功提示
- 带回物品统计
- 收集宠物统计
- 游戏时长

**交互**：
- 任意键/点击返回准备界面

**数据处理**：
- 背包物品转移至仓库
- 宠物捕获记录保存
- 经验/货币结算
- 自动存档

**转场**：
- 任意操作 → 准备界面

#### 5.2.6 撤离失败（DeathScreen）

**入口**：
- 玩家死亡

**内容**：
- 撤离失败提示
- 损失统计（丢失物品）
- 宠物是否丢失（根据死亡原因）

**交互**：
- 任意键/点击返回准备界面

**数据处理**：
- 背包物品清空
- 仓库保留
- 宠物保留

**转场**：
- 任意操作 → 准备界面
